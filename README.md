# Hospital Management System

This is our Database Management Systems (DBMS) semester project. We developed a Hospital Management System using PostgreSQL, Node.js, Express.js, HTML, CSS and JavaScript. The system is designed to manage different hospital operations like patients, doctors, appointments, billing, medicines and medical records.

## Features

- Manage Patients
- Manage Doctors
- Manage Appointments
- Manage Departments
- Manage Rooms and Beds
- Manage Medicines
- Manage Medical Records
- Manage Laboratory Tests
- Manage Prescriptions
- Manage Bills and Payments
- Manage Staff and Nurses
- Manage Insurance Policies
- Dashboard with hospital statistics and reports

## How to Run

Clone the repository

```bash
git clone https://github.com/areeb-shariq/Hospital-Management-System.git
```

Go to the API folder

```bash
cd API
```

Install the required packages

```bash
npm install
```

Create a `.env` file and add your PostgreSQL database credentials.

Start the server

```bash
node dbms.js
```

Open `CLIENT/index.html` in your browser.
